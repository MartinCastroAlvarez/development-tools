# Android Tools

Build android application.
```bash
android.build
```

Build application keystore.
```bash
android.keysore <alias>
```

Retrieve device logs.
```bash
android.logs
```

Run unit tests.
```bash
android.unit
```

Deploy android application.
```bash
android.deploy
```

Lint android code.
```bash
android.lint
```

Run instrumented tests.
```bash
android.qa
```
