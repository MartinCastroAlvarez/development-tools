# Docker Tools

Build Docker image:
```bash
docker.build <tag>
```

Push Docker image:
```bash
docker.push <tag>
```

Login into one of the containers:
```bash
docker.ssh <container-id>
```

Clean cached Docker images:
```bash
docker.clean
```

Start Cassandra cluster:
```bash
docker.container
```

Start Clasticsearch cluster:
```bash
docker.elastic
```

Start MySQL server:
```bash
docker.sql
```

Start PostgreSQL server:
```bash
docker.pysql
```

Start Spark cluster:
```bash
docker.spark
```

Start Redis cluster:
```bash
docker.redis
```
